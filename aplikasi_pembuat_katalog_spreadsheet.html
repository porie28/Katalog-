Cadangkan & Impor Data Manual (.json)

Ingin memindahkan seluruh katalog ke perangkat lain secara offline? Gunakan fitur ekspor/impor file data di bawah ini.

Tips Berbagi Katalog yang Sukses:

  1. Gunakan URL Gambar: Saat membagikan via link, pastikan produk menggunakan metode "Link URL Gambar". Gambar yang diupload dari HP pribadi (lokal) memiliki ukuran data yang sangat besar sehingga tidak bisa dikompresi ke dalam link URL.
  2. Uji Coba Kirim: Klik tombol "Bagikan Link" di kanan atas, salin link tersebut, lalu coba buka di tab baru browser atau kirimkan ke HP Anda sendiri untuk melihat hasilnya.

Bagikan Katalog Anda!

Salin link di bawah ini dan kirimkan ke rekan, pelanggan, atau simpan sendiri sebagai tautan instan.

PENTING: Anda membuka aplikasi sebagai file lokal!

Link di bawah ini mengandung jalur lokal (file:///C:...). Link ini tidak akan bisa dibuka di perangkat lain karena file ini hanya ada di komputer Anda.

Pilih salah satu cara agar bisa dibagikan:

  • Cara Mudah: Gunakan fitur "Unduh File Data (.json)" di halaman utama lalu kirim file tersebut lewat WhatsApp beserta file HTML ini. Rekan Anda cukup mengunggahnya.
  • Cara Online: Unggah file HTML ini ke layanan web hosting gratis seperti tiiny.host atau GitHub Pages agar memiliki link https://.

Catatan Gambar: Beberapa produk Anda menggunakan file gambar lokal dari HP/PC. Gambar lokal tidak ikut terbawa dalam link sharing ini. Gunakan "Link URL Gambar" jika gambar ingin terlihat oleh orang lain.
