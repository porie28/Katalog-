Koneksi Otomatis Google Sheets Aktif!

Sistem ini terintegrasi langsung menggunakan Google Apps Script. Tambahkan produk baru melalui tombol di bawah, dan data Anda akan langsung terkirim dan tersimpan rapi di spreadsheet tanpa ketik ulang maupun copy-paste manual.

⚙️ Konfigurasi Apps Script (Hanya Perlu Dilakukan Sekali)

Langkah 1: Buka Google Sheets target Anda.

Langkah 2: Klik menu Ekstensi / Extensions di bagian atas → klik Apps Script.

Langkah 3: Hapus kode bawaan yang kosong, paste kode hijau di samping kanan ini.

Langkah 4: Klik Simpan 💾.

Langkah 5: Klik Terapkan (Deploy) → pilih Penerapan Baru (New Deployment).

Langkah 6: Klik tombol gerigi di sebelah kiri 'Pilih Jenis' → Pilih Aplikasi Web / Web App.

Langkah 7: Ubah pengaturan Yang memiliki akses menjadi "Siapa saja" / "Anyone".

Langkah 8: Jalankan deploy, berikan izin Google Anda, lalu salin URL Aplikasi Web yang didapatkan ke kolom Panel 2 di bawah.

Google Apps Script Code

1. Tautan Pembacaan Katalog (CSV)

Masukkan URL hasil Publikasikan CSV Google Sheets Anda untuk menampilkan daftar barang ke web.

2. Tautan Kirim Otomatis (Apps Script URL)

Masukkan alamat URL Aplikasi Web (Web App URL) dari hasil Deploy Apps Script Anda.

3. Aksi Tambah Produk Baru

Begitu Anda mengisi data produk baru di sini, data akan langsung diposting secara instan ke spreadsheet.

⚠️ Harap hubungkan Tautan Kirim di Panel 2!
