PENTING: Anda membuka aplikasi sebagai file lokal!

Link di bawah ini mengandung jalur lokal (file:///C:...). Link ini tidak akan bisa dibuka di perangkat lain karena file ini hanya ada di komputer Anda.

Pilih salah satu cara agar bisa dibagikan:

  • Cara Mudah: Gunakan fitur "Unduh File Data (.json)" di halaman utama lalu kirim file tersebut lewat WhatsApp beserta file HTML ini. Rekan Anda cukup mengunggahnya.
  • Cara Online: Unggah file HTML ini ke layanan web hosting gratis seperti **Netlify** atau **GitHub Pages** agar memiliki link https://.

Catatan Gambar: Beberapa produk Anda masih menggunakan gambar lokal tanpa kompresi maksimal. Gunakan tombol "Upload ke Cloud" agar gambar stabil tampil di WA orang lain.

Panduan Hosting Gratis & Permanen (Selamanya)

Berikut adalah dua layanan paling populer dan 100% gratis selamanya untuk menyimpan file aplikasi ini di internet.

METODE 1

Netlify Drop (Paling Mudah, Drag & Drop)

Sangat direkomendasikan jika Anda ingin cepat dan tidak mau ribet menulis kode.

  1. Buat folder baru di komputer Anda, masukkan file index.html (file aplikasi ini) ke dalam folder tersebut.
  2. Buka situs app.netlify.com/drop di browser.
  3. Daftar akun gratis Netlify menggunakan Email atau Google (100% gratis selamanya).
  4. Seret (drag & drop) folder tersebut ke dalam kotak yang tersedia di situs Netlify.
  5. Situs Anda langsung online selamanya dengan alamat link berakhiran .netlify.app!
METODE 2

GitHub Pages (Sangat Stabil & Profesional)

Layanan resmi gratis milik Microsoft GitHub, sangat populer digunakan developer.

  1. Buka GitHub.com dan daftarkan akun baru secara gratis.
  2. Buat Repository baru dengan nama bebas (misal: katalogku). Pilih opsi Public.
  3. Klik tulisan "uploading an existing file" pada layar awal repositori.
  4. Seret file index.html Anda ke sana dan klik Commit changes.
  5. Pergi ke menu Settings (ikon roda gigi) -> pilih submenu Pages di sisi kiri.
  6. Di bagian Build and deployment, ganti pilihan None menjadi main (atau master), lalu klik Save.
  7. Tunggu 1 menit, web Anda online di alamat: https://username-anda.github.io/katalogku/.

Panduan Kolaborasi Google Sheets

Agar data Google Sheets dapat terbaca sempurna oleh aplikasi, buatlah kolom tabel Anda dengan susunan urutan judul baris pertama (Header) sebagai berikut:

Nama Kategori Harga Deskripsi Gambar Stok
Disket Komputer 5000 Memori Kuno https://... 20

Langkah mempublikasikannya menjadi CSV:

  1. Di Google Sheets, klik menu File -> pilih Bagikan (Share) -> klik Publikasikan ke Web (Publish to web).
  2. Ubah format dari Seluruh Dokumen (Web Page) menjadi Nilai yang dipisahkan koma (.csv).
  3. Klik tombol Publikasikan dan konfirmasi.
  4. Salin tautan/link URL yang muncul lalu tempelkan ke kolom koneksi di aplikasi.
